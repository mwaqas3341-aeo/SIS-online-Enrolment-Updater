ℹ️ About This Dashboard

This dashboard is a read-only mirror of publicly accessible data from sis.pesrp.edu.pk — the official School Information System of the Punjab Education Sector Reform Programme. Data is scraped every 6 hours via GitHub Actions and served as a static JSON file on GitHub Pages.

Coverage: E-Transfer status, official notices, district list, and public enrollment statistics. Data is self-reported by schools and is provisional in nature. For authoritative records, visit the official SIS portal.
